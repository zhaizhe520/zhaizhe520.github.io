│
▼
Router (路由层) –> 解析 URL (如 /api/posts/1)
│
▼
Controller (控制器) –> 校验前端传过来的参数 (如验证 ID 是否为数字)
│
▼
Service (业务逻辑) –> 核心逻辑，处理父子表的关联
│
▼
Model/DAO (数据层) –> 执行 SQL 语句 (如 JOIN 查询)
