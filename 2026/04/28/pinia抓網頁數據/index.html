pinia抓網頁數據

抓取地址

http://110.42.248.8/wp-json/wp/v2/posts
這整串是什麼？
這是 WordPress 自帶的官方 REST API 接口地址專門用來抓 WordPress 網站的文章列表
逐段拆解
http://110.42.248.8你 WordPress 伺服器的公網 IP 位址
/wp-json/WordPress 固定前綴，只要是 WP 接口，一定帶這個
/wp/v2/WordPress REST API 的 版本 2
/posts固定路由：拿文章列表

抓取

await axios.get(這個網址)

結構

文章索引規則（重要）

WordPress 給你的文章結構（你用 axios 抓下來會長這樣）
你用 GET /wp-json/wp/v2/posts 抓下來的第一篇文章，結構大概是這樣（我只講關鍵欄位）:

開始抓

  1. 標準請求地址（獲取文章列表）
  1. 獲取特色圖片專用地址（必須加 ?_embed）

細分

  1. 文章標題
  1. 文章內容（完整 HTML，支持 v-html）
  1. 文章段落提取（純文字）
  1. 特色圖片（最穩、不報錯寫法）
  1. 文章標籤（Tags）
