本地部署Wordpress實現Vue3接口前後端分離

一、先给你讲清楚：什么是前后端分离（超简单）

WordPress = 后端 / 数据中心
存文章
存图片
存用户
自动生成 API 接口（不用你写）
Vue3 = 前端 / 展示页面
做界面
做交互
做样式
通过 API 去 WordPress 拿数据
它们的关系：
Vue ← API ← WordPress

二、你本地已经具备的条件（全部满足）

✅ 本地 WordPress 能跑（你已经登录后台了）
✅ Vue3 项目能跑、能打包
✅ 数据库 MySQL 你已经会进了
✅ 你只差：Vue 调用 WP API

  1. 找到 WordPress 自带的 API 地址
    不用插件、不用配置自带接口例如：
    plaintext
    http://localhost/你的WP地址/wp-json/wp/v2/posts
    打开就是文章列表 JSON
  2. Vue 安装 axios（发送请求）
    plaintext
    npm install axios
  3. Vue 里写 3 行代码请求 WP 数据
    js
    axios.get(‘http://localhost/.../posts').then(res => {
    console.log(res.data)
    })
  4. 把数据渲染到 Vue 页面
